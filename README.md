# dotfiles
Environment config files

## Shell Prompt

- Install [Oh My Zsh](https://ohmyz.sh) and the [Powerlevel10k](https://github.com/romkatv/powerlevel10k) theme.
- Run `./install.sh` to place the symlinks in your home directory.
- Open a new terminal (or `source ~/.zshrc`) to load the prompt with the configured plugins and theme.
